### 🐦 Twitter / X（简洁版，适合转推）

我开发了一个**个人知识图谱构建与管理 Agent**。

每天和 AI 对话产生大量经验，但说完就没了。这个项目：
- 📝 自动把对话变成结构化笔记
- 🔍 用大白话就能搜到（不用记文件名）
- 🕸️ 自动发现笔记间的关联，形成知识图谱

开源 + 本地运行 + 不绑定任何平台。试了一下，86 篇笔记 2.7 秒索引完。

github.com/Zola-ops/kb-system

### 🐦 Twitter / X（技术版，适合技术圈）

我开发了一个本地知识管理 Agent。

技术栈：Python + FastAPI + SQLite + sentence-transformers + D3.js

核心设计：
- 外部 LLM 分类对话并生成笔记（不让 Agent 自己评自己）
- 384 维多语言向量做语义搜索
- wikilink + 标签 + 语义相似度三策略自动建图
- MCP 通用协议，Claude/Cursor/Hermes 都能接

17 个测试，代码 1800 行，MIT 开源。

github.com/Zola-ops/kb-system
